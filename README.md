# new_19oclock

### 프로젝트 소개
``` 
> 진행 기간 | 2025년 04월 ~ 2025년 06월 

> 프로젝트 이름 | 19시:일구하고시퍼
	프리랜서와 클라이언트를 연결하는 온라인 마켓. 디자인, 개발, 마케팅, 번역 등 다양한 분야의 전문가 서비스를 거래할 수 있다. 
	
> 프로젝트 설명
	과거 학원 수강 중 팀원들과 개발했던 웹 서비스를 기반으로, 구조적 개선 및 기술 리팩토링을 진행한 개인 프로젝트
      
> 개발 목표 및 주안점
	1. 코드 중복 제거 및 인터페이스 기반 설계 적용
	2. DAO 중심 구조 → JPA 기반 ORM 구조로 전환
	3. JSP → Mustache 템플릿 엔진으로 뷰 레이어 경량화
	4. 전통적인 MVC 방식 → RESTful API 기반 구조로 변경하여 프론트/백엔드 분리 및 확장성 강화
	5. 실시간 채팅 기능 구현 
      
> 개발 환경
  형상관리 : Github
  개발도구 : IntelliJ IDEA
  빌드도구 : Gradle
  DB : H2(개발용 인메모리 DB), JPA
  개발언어 : Java(jdk17), JavaScript, jQuery, HTML5, CSS3
  템플릿 엔진 : Mustache
  서버 : Spring Boot 2.1.9 내장 Tomcat (v9.0.24)
      
```

### 기능 소개
``` 
> 서비스 이용자(의뢰인)
    1. 서비스 조회
     - 카테고리를 통한 서비스 조회
     - 검색 키워드를 통한 서비스 검색
     - 서비스 세부 페이지 확인
    2. 찜 하기
     - 마음에 들거나 관심있는 서비스를 저장하는 기능
    3. 의뢰글 작성
     - 원하는 서비스가 없으면 직접 의뢰글을 작성
    4. 결제
    - 결정된 서비스를 결제
    5. 채팅
     - 프리랜서에게 문의할 내용 또는 작업 진행 내용을 전달할 때 채팅 이용
    6. 이용후기/평점
     - 거래가 끝나면 해당 프리랜서 게시글에 이용후기/평점을 등록

> 서비스 판매자(프리랜서)
    1. 서비스 글 작성
      - 제목,카테고리, 지역,가격, 작업기간 등 내용을 기입
      - 등록을 하면 관리자에게 업로드 요청
    2. 알림 기능
      - 사용자에게 채택된 경우 알림
      - 관리자의 서비스 승인 알림 메시지
      - 의뢰인의 게시글에 후기가 등록된 경우
    3. 채팅 기능
      - 의뢰인과 정보를 소통할 수 있도록 채팅 기능
    4. 마이페이지
      - 프리랜서 자기소개 관리
      - 작성한 서비스 글 관리

> 사이트 관리자
    1. 이용자 관리(의뢰인 관리)
      - 프리랜서들의 평가를 기준으로 좋지 않은 이용자를 관리
      - 회원에게 메시지 
    2. 제공자 관리(프리랜서 관리)
      - 프리랜서가 서비스 등록 하면 관리자가 확인하고 승인
    3. 게시판 관리
      - 공지사항 게시글 작성/수정/삭제
      - 고객문의 문의글 답변 작성/수정/삭제 기능
      
```
